# Code of Conduct

reflector is a research-oriented open-source project. Contributors are expected to keep discussion respectful, technically grounded, and constructive.

## Expected behavior

- engage with curiosity, clarity, and professionalism
- focus on evidence, reproducibility, and reviewable changes
- assume good faith while still making technical disagreements explicit
- help keep the repository accessible to contributors from different backgrounds and experience levels

## Unacceptable behavior

- harassment, intimidation, or personal attacks
- discriminatory or exclusionary language
- repeated bad-faith disruption of technical review or collaboration
- sharing sensitive information without permission

## Reporting

If you experience or witness behavior that violates this code of conduct, contact the maintainer through GitHub. For non-sensitive issues, open an issue so the concern can be addressed transparently; for sensitive matters, report privately.
