# Purpose

This specification defines a reusable blueprint for generating holistic infographic-style visual artifacts from complex source material.

The goal is NOT merely summarization.

The goal is:
- conceptual compression
- emotional digestibility
- visual synthesis
- cognitive accessibility
- artistic meaning extraction

The output should feel like:
> a holistic lens into an idea.

This specification is intended to work across:
- research papers
- conversations
- therapy reflections
- technical systems
- philosophical concepts
- creative ideas
- repositories
- architectures
- emotional insights
- educational material

---

# Core Philosophy

The infographic should transform dense or abstract information into:
- digestible structure
- emotionally approachable visuals
- intuitive conceptual hierarchy
- aesthetically coherent educational art

The artifact should:
- reduce cognitive overload
- preserve nuance
- communicate systems-level relationships
- encourage curiosity
- reinforce understanding visually

The infographic is NOT intended to:
- overwhelm with information
- replicate the source material verbatim
- maximize technical density
- behave like a slide deck
- behave like raw notes

Instead, it should:
> compress meaning into visual understanding.

---

# Guiding Principles

## 1. Holistic Compression

Extract:
- the most meaningful concepts
- the highest-level patterns
- the strongest relationships
- the emotional or philosophical core
- the systems-level implications

Avoid:
- excessive detail
- repetitive information
- overly granular explanations
- exhaustive completeness

The infographic should prioritize:
> understanding over coverage.

---

## 2. Cognitive Accessibility

The infographic should be:
- visually scannable
- easy to follow
- emotionally lightweight
- naturally grouped
- hierarchy-driven

The viewer should be able to:
- understand the high-level idea quickly
- progressively explore deeper concepts
- intuit relationships visually

---

## 3. Visual Hierarchy

Use:
- major title areas
- grouped concept panels
- modular sections
- visual separators
- iconography
- diagrams
- arrows
- symbolic metaphors
- flow relationships

Balance:
- whitespace
- visual density
- readability
- aesthetics

Avoid:
- clutter
- excessive text walls
- overwhelming diagrams
- hyper-technical layouts

---

## 4. Emotional Resonance

The infographic should feel:
- thoughtful
- beautiful
- reflective
- intelligent
- calming
- inspiring
- futuristic
- human

Even technical concepts should retain:
- warmth
- humanity
- curiosity
- creative energy

---

## 5. Conceptual Structure

Most infographic outputs should naturally contain some combination of:

- overview / definition
- core concepts
- system relationships
- workflow or process
- major insights
- tensions or tradeoffs
- human implications
- future implications
- ethical considerations
- practical applications
- reflective takeaway

Not all sections are required.

The structure should emerge naturally from:
- the source material
- the scope
- the complexity
- the emotional tone

---

# Source Material Handling

The source may include:
- PDFs
- conversations
- markdown
- repositories
- screenshots
- technical notes
- journals
- transcripts
- research papers
- philosophical writing

The system should:
- identify coherent themes
- extract scoped insight clusters
- avoid mixing unrelated concepts
- maintain conceptual clarity

One infographic should generally focus on:
> one primary conceptual cluster.

Avoid trying to compress an entire universe into one artifact.

---

# Insight Extraction Rules

The infographic should prioritize:
- systems-level ideas
- conceptual relationships
- recurring themes
- useful metaphors
- practical implications
- emotionally resonant insights

The system may:
- generalize private details
- abstract personal experiences
- synthesize patterns
- condense multiple related ideas

The system should avoid:
- exposing sensitive personal information
- unnecessary specificity
- emotionally unsafe extraction
- raw emotional dumping
- oversharing identifiable details

- a modern academic poster

Possible visual elements:
- icons
- nodes

- subtle decorative textures

# Aesthetic Separation

This specification defines:
- cognitive structure
- compression philosophy
- infographic behavior

It does NOT define:
- specific art direction
- colors
- rendering style
- typography choices
- visual genre

These are delegated to:
> separate style specifications.

This separation allows:
- one conceptual system
- many aesthetic renderings

Examples:
- futuristic tech
- feminine soft design
- cybernetic
- therapeutic
- academic
- anime-inspired
- startup infographic
- README technical
- minimalist

The final infographic should feel:
- cohesive

- visually digestible
