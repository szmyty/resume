# visual-synapse.spec.md

## Purpose
Coordinate semantic and visual artifacts in recursive publication workflows.

## Visual Synapse Definition
A visual synapse is a synchronization link between:
- semantic intent (section argument)
- visual representation (figure/diagram)
- publication placement (caption, label, reference)

## Synapse Rules
- Every figure referenced in manuscript text must have a canonical caption entry.
- Diagram updates require a paired semantic review in the same cycle.
- Captions must declare the synchronization concept being demonstrated.

## Drift Signals
- orphaned figure reference
- caption without manuscript reference
- semantic claim changed without diagram refresh

## Checkpoint Expectation
Visual synapse drift must be resolved before a publication checkpoint can pass.
