# Support

## Getting help

For most questions, start with the repository documentation:

- [`README.md`](./README.md)
- [`00-README.md`](./00-README.md)
- [`CONTRIBUTING.md`](./CONTRIBUTING.md)
- [`docs/`](./docs/)

## Where to ask questions

- Open an issue for bugs, documentation gaps, or reproducible workflow problems.
- Open an issue for research or architecture questions when a repository-linked answer would help future contributors.

## Before opening an issue

Please include:

- what you were trying to do
- the command or workflow you ran
- the files or specifications involved
- any relevant logs or error messages
